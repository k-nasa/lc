## Installation

### Pre-compiled executables

Get them [here](https://github.com/k-nasa/mlc/releases)

### using homebrew

```
brew install k-nasa/tap/mlc
```

### using cargo

Currently it cannot be built with the stable version.

```
cargo install mlc
```
